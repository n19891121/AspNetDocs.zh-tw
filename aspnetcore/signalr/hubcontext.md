### <a name="inject-a-strongly-typed-hubcontext"></a><span data-ttu-id="4eaf7-122">插入的強型別 HubContext</span><span class="sxs-lookup"><span data-stu-id="4eaf7-122">Inject a strongly-typed HubContext</span></span>

<span data-ttu-id="4eaf7-123">若要插入的強型別 HubContext，請確定您的中樞繼承自`Hub<T>`。</span><span class="sxs-lookup"><span data-stu-id="4eaf7-123">To inject a strongly-typed HubContext, ensure your Hub inherits from `Hub<T>`.</span></span> <span data-ttu-id="4eaf7-124">將使用其插入`IHubContext<THub, T>`介面而非`IHubContext<THub>`。</span><span class="sxs-lookup"><span data-stu-id="4eaf7-124">Inject it using the `IHubContext<THub, T>` interface rather than `IHubContext<THub>`.</span></span>

```csharp
public class ChatController : Controller
{
    public IHubContext<ChatHub, IChatClient> _strongChatHubContext { get; }

    public ChatController(IHubContext<ChatHub, IChatClient> chatHubContext)
    {
        _strongChatHubContext = chatHubContext;
    }

    public async Task SendMessage(string message)
    {
        await _strongChatHubContext.Clients.All.ReceiveMessage(message);
    }
}
```
