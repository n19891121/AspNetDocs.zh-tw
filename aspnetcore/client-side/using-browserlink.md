## <a name="how-it-works"></a><span data-ttu-id="d31de-159">它的運作方式</span><span class="sxs-lookup"><span data-stu-id="d31de-159">How it works</span></span>

<span data-ttu-id="d31de-160">瀏覽器連結會使用 SignalR 來建立 Visual Studio 和瀏覽器之間的通訊通道。</span><span class="sxs-lookup"><span data-stu-id="d31de-160">Browser Link uses SignalR to create a communication channel between Visual Studio and the browser.</span></span> <span data-ttu-id="d31de-161">啟用瀏覽器連結時，Visual Studio 就會作為多個用戶端 （瀏覽器） 可以連線到 SignalR 伺服器。</span><span class="sxs-lookup"><span data-stu-id="d31de-161">When Browser Link is enabled, Visual Studio acts as a SignalR server that multiple clients (browsers) can connect to.</span></span> <span data-ttu-id="d31de-162">瀏覽器連結也會註冊在 ASP.NET Core 要求管線的中介軟體元件。</span><span class="sxs-lookup"><span data-stu-id="d31de-162">Browser Link also registers a middleware component in the ASP.NET Core request pipeline.</span></span> <span data-ttu-id="d31de-163">此元件會插入特殊`<script>`到每一個網頁要求從伺服器中的參考。</span><span class="sxs-lookup"><span data-stu-id="d31de-163">This component injects special `<script>` references into every page request from the server.</span></span> <span data-ttu-id="d31de-164">您可以選取，以查看指令碼參考**檢視原始檔**在瀏覽器，並向下捲動到結尾`<body>`標記的內容：</span><span class="sxs-lookup"><span data-stu-id="d31de-164">You can see the script references by selecting **View source** in the browser and scrolling to the end of the `<body>` tag content:</span></span>

```html
    <!-- Visual Studio Browser Link -->
    <script type="application/json" id="__browserLink_initializationData">
        {"requestId":"a717d5a07c1741949a7cefd6fa2bad08","requestMappingFromServer":false}
    </script>
    <script type="text/javascript" src="http://localhost:54139/b6e36e429d034f578ebccd6a79bf19bf/browserLink" async="async"></script>
    <!-- End Browser Link -->
</body>
```

<span data-ttu-id="d31de-165">您的原始程式檔不會被修改。</span><span class="sxs-lookup"><span data-stu-id="d31de-165">Your source files aren't modified.</span></span> <span data-ttu-id="d31de-166">中介層元件會以動態方式插入指令碼參考。</span><span class="sxs-lookup"><span data-stu-id="d31de-166">The middleware component injects the script references dynamically.</span></span>

<span data-ttu-id="d31de-167">由於瀏覽器端的程式碼全部都是 JavaScript，它適用於 SignalR 支援的所有瀏覽器，而不需要瀏覽器外掛程式。</span><span class="sxs-lookup"><span data-stu-id="d31de-167">Because the browser-side code is all JavaScript, it works on all browsers that SignalR supports without requiring a browser plug-in.</span></span>
