<a name="the-effects-of-caching"></a>快取的效果
====================
由 Eric Lee

現今幾乎所有 Web 應用程式都需要存取資料，從資料庫中，但這個常見的作業會對應用程式的整體效能的影響力。 這段影片會說明 Visual Studio Team System 如何協助您識別您的應用程式冗餘的呼叫至資料庫，並接著示範您如何運用 ASP.NET 2.0 資料來源快取功能以消除冗餘的呼叫。
