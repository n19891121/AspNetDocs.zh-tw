# <a name="using-the-load-test-agent"></a>使用負載測試代理程式

Eric 先生

在這段影片中，我們將探討如何將負載測試代理程式併入我們使用 Visual Studio Team System 進行的測試。 負載測試代理程式是「rig」的一部分，可以部署到不同的電腦，以便從另一個作業系統或電腦設定進行負載、壓力和單元測試。 每個 rig 都可以與數個代理程式搭配使用，而且可以部署到虛擬或遠端電腦，以 Visual Studio Team System 進行全面的測試。
