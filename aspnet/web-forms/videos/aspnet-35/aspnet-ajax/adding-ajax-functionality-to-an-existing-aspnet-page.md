# <a name="adding-ajax-functionality-to-an-existing-aspnet-page"></a>將 AJAX 功能新增至現有的 ASP.NET 頁面

依[Todd Miranda](https://github.com/tmiranda)

在 ASP.NET 網頁中使用 AJAX 可以提升效能，並提升應用程式的整體使用者體驗。 您不需要重寫現有的頁面，即可加入 AJAX 功能。 只要修改幾次，就可以使用 AJAX 來建立現有的 ASP.NET 網頁。 這段影片示範如何將 AJAX 功能以最少的方式新增至現有的 ASP.NET 網頁。
