### <a name="how-the-membership-framework-handles-invalid-login-attempts"></a><span data-ttu-id="e5768-150">成員資格架構如何處理無效的登入嘗試</span><span class="sxs-lookup"><span data-stu-id="e5768-150">How the Membership Framework Handles Invalid Login Attempts</span></span>

<span data-ttu-id="e5768-151">當訪客達到登入頁面，並提交其認證時，則其瀏覽器會將 HTTP 要求對登入頁面。</span><span class="sxs-lookup"><span data-stu-id="e5768-151">When a visitor reaches the login page and submits their credentials, their browser makes an HTTP request to the login page.</span></span> <span data-ttu-id="e5768-152">如果認證有效，則 HTTP 回應會包含驗證票證在 cookie 中。</span><span class="sxs-lookup"><span data-stu-id="e5768-152">If the credentials are valid, the HTTP response includes the authentication ticket in a cookie.</span></span> <span data-ttu-id="e5768-153">因此，駭客嘗試入侵您的站台可以建立可徹底登入頁面，以有效的使用者名稱和密碼猜出傳送 HTTP 要求的程式。</span><span class="sxs-lookup"><span data-stu-id="e5768-153">Therefore, a hacker attempting to break into your site could create a program that exhaustively sends HTTP requests to the login page with a valid username and a guess at the password.</span></span> <span data-ttu-id="e5768-154">如果密碼猜測值正確無誤，請登入頁面就會傳回驗證票證 cookie，此時程式知道它有偶然的有效使用者名稱/密碼組。</span><span class="sxs-lookup"><span data-stu-id="e5768-154">If the password guess is correct, the login page will return the authentication ticket cookie, at which point the program knows it has stumbled upon a valid username/password pair.</span></span> <span data-ttu-id="e5768-155">透過暴力密碼破解，只有弱式密碼時，尤其是可能能夠樂使用者的密碼，這類程式。</span><span class="sxs-lookup"><span data-stu-id="e5768-155">Through brute force, such a program might be able to stumble upon a user's password, especially if the password is weak.</span></span>

<span data-ttu-id="e5768-156">若要避免這類的暴力密碼破解攻擊，成員資格架構鎖定使用者如果有一段時間內的登入失敗嘗試數目。</span><span class="sxs-lookup"><span data-stu-id="e5768-156">To prevent such brute force attacks, the Membership framework locks out a user if there are a certain number of unsuccessful login attempts within a certain period of time.</span></span> <span data-ttu-id="e5768-157">確切的參數是可透過下列兩個成員資格提供者組態設定進行設定：</span><span class="sxs-lookup"><span data-stu-id="e5768-157">The exact parameters are configurable through the following two Membership provider configuration settings:</span></span>
