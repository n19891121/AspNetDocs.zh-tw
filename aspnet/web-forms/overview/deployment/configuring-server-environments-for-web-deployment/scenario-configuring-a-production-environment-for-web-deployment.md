- <span data-ttu-id="36d69-111">環境是由多個負載平衡的 web 伺服器和一或多個資料庫伺服器，通常與容錯移轉叢集和資料庫鏡像所組成。</span><span class="sxs-lookup"><span data-stu-id="36d69-111">The environment consists of multiple load-balanced web servers and one or more database servers, often with failover clustering and database mirroring.</span></span>
- <span data-ttu-id="36d69-112">如果環境是網際網路對向，它可能是從內部部署網路隔離。</span><span class="sxs-lookup"><span data-stu-id="36d69-112">If the environment is Internet-facing, it's likely to be segregated from your internal network.</span></span> <span data-ttu-id="36d69-113">它可能是不同的子網路周邊網路中，可能在不同網域中，而且可能是完全不同的網路基礎結構上。</span><span class="sxs-lookup"><span data-stu-id="36d69-113">It may be on a different subnet in a perimeter network, it may be on a different domain, and it may be on an entirely different network infrastructure.</span></span>
