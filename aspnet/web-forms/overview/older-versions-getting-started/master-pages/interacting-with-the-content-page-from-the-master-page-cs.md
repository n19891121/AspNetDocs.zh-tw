<span data-ttu-id="dd673-127">這個相同的模式可用來在其內容頁面的主版頁面觸發程序功能：</span><span class="sxs-lookup"><span data-stu-id="dd673-127">This same pattern can be used to have a master page trigger functionality in its content pages:</span></span>
