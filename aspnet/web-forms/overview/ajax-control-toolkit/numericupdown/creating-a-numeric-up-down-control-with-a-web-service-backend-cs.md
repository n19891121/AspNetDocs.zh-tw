## <a name="overview"></a>總覽

而不是讓使用者核取方塊中輸入值，數值上下按鈕控制項 （也就存在於 Windows 和其他作業系統） 可以證明越多習慣。 根據預設，`NumericUpDown`控制項一律會增加或減少值 1，但 web 服務證明更大的彈性。
