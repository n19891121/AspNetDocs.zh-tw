> 動畫控制項在 ASP.NET AJAX Control Toolkit 中不只是控制項，但若要將動畫加入至控制項的整個架構。 一般而言，啟動動畫時觸發相同的控制項與使用者互動。 不過也可以使用一個控制項，然後動畫互動另一個控制項。


## <a name="overview"></a>總覽

動畫控制項在 ASP.NET AJAX Control Toolkit 中不只是控制項，但若要將動畫加入至控制項的整個架構。 一般而言，啟動動畫時觸發相同的控制項與使用者互動。 不過也可以使用一個控制項，然後動畫互動另一個控制項。
