> 在 AJAX Control Toolkit reorderlist 的回傳控制項提供使用者透過拖放來重新排列的清單。 已重新排列清單，每當回傳會通知變更的伺服器。

## <a name="overview"></a>總覽

`ReorderList` AJAX Control Toolkit 中的控制項提供使用者透過拖放來重新排列的清單。 已重新排列清單，每當回傳會通知變更的伺服器。
