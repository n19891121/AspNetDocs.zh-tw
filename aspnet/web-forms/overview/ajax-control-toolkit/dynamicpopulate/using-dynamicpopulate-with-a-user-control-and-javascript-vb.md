## <a name="overview"></a>總覽

`DynamicPopulate`中 ASP.NET AJAX Control Toolkit 控制項呼叫 web 服務 （或頁面方法），並會產生的值填滿到目標控制項在頁面上，而不需要重新整理頁面。 它也可觸發程序使用自訂用戶端 JavaScript 程式碼的母體擴展。 不過，特別注意具有擴充項位於使用者控制項時所要採取。
