<span data-ttu-id="79870-126">快取可大幅提升應用程式 s 的整體效能和延展性會耗費大量資源來產生資料並將其複本儲存在可以更有效率地存取的位置。</span><span class="sxs-lookup"><span data-stu-id="79870-126">Caching can greatly improve an application s overall performance and scalability by taking data that is expensive to generate and storing a copy of it in a location that can be more efficiently accessed.</span></span> <span data-ttu-id="79870-127">因為快取會儲存只是實際的基礎資料的副本，它也可能會過期，或*過時*，如果基礎資料變更。</span><span class="sxs-lookup"><span data-stu-id="79870-127">Since the cache holds just a copy of the actual, underlying data, it can become outdated, or *stale*, if the underlying data changes.</span></span> <span data-ttu-id="79870-128">為了克服這點，為網頁開發人員可以指出快取項目將會用的準則*收回*從快取中，使用：</span><span class="sxs-lookup"><span data-stu-id="79870-128">To combat this, a page developer can indicate criteria by which the cache item will be *evicted* from the cache, using either:</span></span>
