## <a name="conclusion"></a>結論

在本教學課程中，您可以建立使用 Entity Framework 與資料庫，並提供在資料層之上的公眾對應介面的 ASP.NET Web API 通訊的應用程式。 我們可以利用 ASP.NET MVC 4 來轉譯 HTML 網頁和 Knockout.js 以及 jQuery 提供動態的互動，而不需要頁面重新載入。
