# <a name="owin-and-katana"></a>OWIN 和 Katana

Katana 是一組有彈性的元件，用於建立和裝載適用于 .NET (OWIN) 型 web 應用程式的開放 Web 介面。 新的開發應該使用 [ASP.NET Core](/aspnet/core/introduction-to-aspnet-core)。 連結 `Microsoft.Owin.*` 庫不適用於 .Net Core，因為 ASP.NET Core 有對等的取代專案。
