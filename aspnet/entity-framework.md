# <a name="entity-framework"></a>Entity Framework

Entity Framework （EF）是物件關聯式對應程式，可讓 .NET 開發人員使用特定領域物件來處理關聯式資料。 有了 Entity Framework，開發人員便不再需要撰寫大多數必須撰寫的資料存取程式碼。

- [取得：](https://msdn.com/data/ee712906)將此新增至您的專案，並開始利用這個強大的 O/RM。
- [開始使用：](https://msdn.com/data/ee712907)請流覽指南，以取得可協助您充分利用資料的影片和教學課程。
- [取得協助：](https://msdn.com/data/hh913619)瞭解如何從專家取得協助，以及如何在 Stack Overflow 上貢獻資料社區。
- [貢獻：](https://github.com/aspnet/EntityFramework6)藉由造訪 GitHub 網站，瞭解如何協助 Entity Framework、開放的開發模型變得更好。
