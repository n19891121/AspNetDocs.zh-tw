<span data-ttu-id="53714-279">Web 開發人員應該繼續以確保其站台不是很容易遭受 XSS 攻擊。</span><span class="sxs-lookup"><span data-stu-id="53714-279">Web developers should continue to ensure that their site is not vulnerable to XSS attacks.</span></span> <span data-ttu-id="53714-280">XSS 攻擊非常強大，而成功的攻擊也會中斷 XSRF 攻擊的防禦 ASP.NET Web 堆疊執行階段。</span><span class="sxs-lookup"><span data-stu-id="53714-280">XSS attacks are very powerful, and a successful exploit would also break the ASP.NET Web Stack Runtime defenses against XSRF attacks.</span></span>
