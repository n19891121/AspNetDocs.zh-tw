- <span data-ttu-id="9f672-119">**模型**。</span><span class="sxs-lookup"><span data-stu-id="9f672-119">**Models**.</span></span> <span data-ttu-id="9f672-120">模型物件是應用程式的部分，可為應用程式的資料網域執行邏輯。</span><span class="sxs-lookup"><span data-stu-id="9f672-120">Model objects are the parts of the application that implement the logic for the application s data domain.</span></span> <span data-ttu-id="9f672-121">通常，模型物件會擷取和儲存資料庫中的模型狀態。</span><span class="sxs-lookup"><span data-stu-id="9f672-121">Often, model objects retrieve and store model state in a database.</span></span> <span data-ttu-id="9f672-122">例如，產品物件可能會從資料庫抓取資訊、對其進行操作，然後將更新的資訊寫回 SQL Server 中的 Products 資料表。</span><span class="sxs-lookup"><span data-stu-id="9f672-122">For example, a Product object might retrieve information from a database, operate on it, and then write updated information back to a Products table in SQL Server.</span></span>
