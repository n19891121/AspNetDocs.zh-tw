# <a name="dynamic-v-strongly-typed-views"></a>動態與 強型別檢視

藉由[Rick Anderson]((https://twitter.com/RickAndMSFT))

有三種方式可將資訊從控制器傳遞至 ASP.NET MVC 3 中的檢視：

1. 為強型別的模型物件。
2. 做為動態類型 (使用@model動態)
3. 使用 ViewBag
