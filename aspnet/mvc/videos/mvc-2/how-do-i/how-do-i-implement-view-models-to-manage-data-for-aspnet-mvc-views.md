# <a name="how-do-i-implement-view--models-to-manage-data-for-aspnet-mvc-views"></a>如何：執行視圖模型來管理 ASP.NET MVC 視圖的資料？

依[Chris Pels](https://twitter.com/chrispels)

在這段影片中，Chris Pels 說明如何使用「視圖模型」的概念來管理 ASP.NET MVC 視圖的資料。 首先，會建立一個範例 MVC 專案，使用 ViewDataDictionary 來封裝，並將資料從控制器傳遞到 view。 接下來，會建立一個類別，代表範例 MVC 專案中視圖的視圖模型。 在此執行過程中，會討論視圖模型方法的優點。 最後，討論如何在使用視圖模型執行時分割應用程式邏輯。
